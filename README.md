# 项目简介

## 1. CodeBuilder:二维码条形码生成器项目

> 用VS2008编码，
> zxing.dll实现生成各种码，
> 插件DXperience-8.3.4生成报表


**主要功能是：**

    - (1) 根据输入的内容生成二维码
    - (2) 根据输入的数字生成条形码
    - (3) 根据选择随机生成条形码
    - (4) 生成的结果以报表的形式显示，并能够打印
